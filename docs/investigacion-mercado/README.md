# Investigación de mercado — Lince

Planillas para **mirar el mercado hacia afuera** (no datos internos de la empresa
ni de clientes concretos). Abrilas en Excel o Google Sheets (CSV, UTF-8).

Cada archivo trae **una fila de ejemplo** (ID `EJ-01`) para mostrar cómo se llena.
Borrala o reemplazala al cargar datos reales.

## Hojas

| Archivo | Para qué sirve |
|---|---|
| `1-demanda-relevamiento.csv` | Lado **demanda**: una fila por negocio que entrevistás. Mide su dolor, herramientas, interés y cuánto pagaría. |
| `2-competencia.csv` | Lado **oferta**: una fila por competidor/proveedor. Qué ofrece, a qué precio, fortalezas y debilidades. |
| `3-preguntas-clave.csv` | Las **preguntas de mercado** que hay que responder y dónde averiguar cada dato. |

## Reglas para no meter datos sensibles
- **No** anotes nombres de personas, teléfonos, emails ni razones sociales.
- Usá **rangos** (ej. "20-40 consultas") en vez de números exactos.
- Para el decisor anotá el **rol** ("dueño", "encargada"), no la persona.
- Identificá cada negocio con un **ID** (EJ-01, EJ-02…), no con su nombre.

## Cómo usarla
1. Empezá por `3-preguntas-clave.csv`: son las hipótesis a validar.
2. Cargá `1-demanda-relevamiento.csv` a medida que entrevistás negocios.
3. Cargá `2-competencia.csv` mirando qué hay en el mercado.
4. Cuando tengas ~10 filas en demanda, vas a ver solo qué rubro y qué precio
   tienen más tracción → ese es tu nicho y tu número de partida.
